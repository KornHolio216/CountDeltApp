# CountDeltApp

> Early vanilla JavaScript math project, kept as part of my learning journey

CountDeltApp is a simple browser-based math application created as an early frontend practice project. It was built with plain HTML, CSS and JavaScript before I started working on more advanced web and machine learning projects.

The app contains a few small tools:

- quadratic equation / delta calculator
- basic calculator
- random number generator

## Features

- calculate the discriminant / delta of a quadratic equation
- display results for quadratic equation calculations
- use a simple calculator interface
- generate random numbers
- navigate between multiple HTML pages
- practice DOM manipulation with vanilla JavaScript

## Technologies

- HTML
- CSS
- JavaScript
